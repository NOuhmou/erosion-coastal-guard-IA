# Prompt pour génération du MCD

**Date:** 16/03/2026
**Outil:** Claude

## Prompt exact :
À partir du modèle de données détaillé ci-dessous (14 entités),
génère un Modèle Conceptuel de Données (MCD) complet avec :

La liste de toutes les entités et leurs attributs
(clés primaires en #, obligatoires en *, FK indiquées)

Les relations entre entités avec les cardinalités (1,1) (0,N) etc.

Un schéma ASCII-art représentant les relations principales

Les contraintes d'intégrité référentielle

Les index recommandés pour les performances

Les règles de gestion (triggers) importantes
